# SecureNoteX

SecureNoteX, kullanıcıların notlarını, görevlerini ve hedeflerini
güvenli bir şekilde yönetebilmesini sağlayan mobil uygulamadır.

## Özellikler
- PIN ile uygulama kilidi
- Not ekleme / silme / güncelleme
- Görev yönetimi (öncelik, tamamlandı durumu)
- Pomodoro destekli görev takibi
- Hedef ekleme ve takip
- Modern ve sade arayüz

## Kullanılan Teknolojiler
- Flutter
- Dart
- Local storage (Hive / SQLite)

## Çalıştırma
```bash
flutter pub get
flutter run
